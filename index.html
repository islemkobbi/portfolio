SelfPlan: Learning Mid-to-Mid Motion Planning through Closed-Map Self-Play

Islem Kobbi, Tiago Rocha Goncalves, Fawzi Nashashibi

Abstract

SelfPlan is a closed-map scalable self-play framework for learning mid-to-mid motion planners. The policy outputs short-horizon trajectories tracked by a low-level controller, using structured ego-frame observations and rewards for path quality, efficiency, safety, and goal reaching.

Motion Planning in Urban Environments via Self-Play Reinforcement Learning

Islem Kobbi, Tiago Rocha Goncalves, Fawzi Nashashibi

Abstract

This work studies self-play reinforcement learning for mid-to-mid motion planning in urban autonomous driving. The planner maps structured observations to continuous trajectories parameterized by cubic Hermite splines while preserving a modular perception-planning-control pipeline.
